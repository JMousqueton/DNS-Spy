What it does

DNS Spy runs 16+ analysis modules in parallel — WHOIS, RDAP, DNS records, SSL/TLS, Geolocation, ASN/BGP, HTTP headers, WAF detection, cookie security, reverse DNS, email security (SPF/DMARC/DKIM), certificate transparency, HTML subdomains, screenshot, tech stack, SaaS detection, and an aggregate security score.
